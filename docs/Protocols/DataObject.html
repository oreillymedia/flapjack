DataObject

public protocol DataObject

An abstraction on top of a model object, typically mapped one-to-one to a database entity such as Core Data’s NSManagedObject or Realm’s Realm.Object. Those conforming to this protocol should be sure to denote the type of PrimaryKey to expose (through the associated type PrimaryKeyType).

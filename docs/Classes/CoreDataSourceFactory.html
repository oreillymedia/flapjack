  • vendObjectsDataSource(attributes:sectionProperty:limit:)

    Creates and returns a data source that will get notified when any objects of a given type are changed, inserted, or deleted so long as it matches the provided attributes. Can be grouped into sections by a sectionProperty, and limited by a limit parameter.

    Declaration

    Swift

    public func vendObjectsDataSource<T>(attributes: DataContext.Attributes, sectionProperty: String?, limit: Int?) -> CoreDataSource<T> where T : NSManagedObject, T : DataObject

    Parameters

    attributes

    An optional dictionary of key-value query constraints to apply to the lookup.
    sectionProperty

    A keypath to a property to use when grouping together results, if desired.
    limit

    An optional limit to be applied to the results.

    Return Value

    A data source that will get notified when any objects matching the attributes are changed, inserted, or deleted.

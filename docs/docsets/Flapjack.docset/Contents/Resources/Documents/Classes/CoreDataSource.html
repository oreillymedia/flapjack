CoreDataSource

public class CoreDataSource<T> : NSObject, NSFetchedResultsControllerDelegate, DataSource where T : NSManagedObject, T : DataObject

Listens for changes in an NSManagedObjectContext based on the NSManagedObjectContextDidChange notification for a series of objects described by a set of attributes. Invokes the onChange clsoure when the objects identified by the lookup change, get deleted, or when new objects are added that match the attributes. Powered by an NSFetchedResultsController.

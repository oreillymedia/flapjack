SingleCoreDataSource

public class SingleCoreDataSource<T> : NSObject, SingleDataSource where T : NSManagedObject, T : DataObject

Listens for changes in an NSManagedObjectContext based on the NSManagedObjectContextDidChange notification for a single object described by a set of attributes. Invokes the onChange clsoure when the monitored object changes in the given DataContext.

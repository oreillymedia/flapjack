DataAccessDelegate

public protocol DataAccessDelegate : AnyObject

An object conforming to the DataAccessDelegate agrees to be notified about events occurring outside the normal call-and-return lifecycle of the methods presented by conformists to DataAccess.

  • dataAccess(_:wantsMigratorForStoreAt:) Default implementation

    Called when the data access object is about to setup the data stack (from on disk into application memory), and if necessary (given the on-disk store at the given URL), should be returned a Migrator object if migrations should be performed. The DataAccess object should check with the provider if migrations should actually be performed, and it’s the Migrator‘s responsibility to deal with the store at the URL to see if it’s up-to-date (for more information, see the documentation for Migrator).

    Default Implementation

    The default implementation of this delegate method returns nil for the Migrator.

    Declaration

    Swift

    func dataAccess(_ dataAccess: DataAccess, wantsMigratorForStoreAt storeURL: URL?) -> Migrator?

    Parameters

    dataAccess

    The object calling this method.
    storeURL

    The on-disk location where the store is located; if there is no on-disk store (like if the store is in-memory only), this will be nil.

    Return Value

    Should return a Migrator object if one needs to be consulted with to begin migrations; nil if not.
